dp分类

1 Max 1D Range Sum

description：Give an integer array A containing n non-zero integers, find the max Range Sum Query(RSQ) between two indices i and j in [0…n-1].

dp[i]代表以A[i]结尾的最大范围和。

对于dp[i]：dp[i]= max(dp[i-1], 0) + A[i].

最大范围和即dp[i]中的最大值。
